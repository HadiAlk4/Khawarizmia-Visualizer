# Khawarizmia Visualizer

An interactive, web-based Data Structures and Algorithms visualizer designed to turn abstract computer science concepts into clear, step-by-step animations. 

 **Live Demo:** Hosted on [Vercel](https://khawarizmia-visualizer.vercel.app)

---

## 📌 Features

### Sorting Algorithms
* **Selection Sort** 
* **Bubble Sort** 

### Advanced Structures & Parsers
* **Prefix & Postfix Calculator**
* **d-ary Heaps** (WiP)
* **AVL Tree** *(Balanced iteratively without recursion)* (WiP)

---

## 🛠️ Tech Stack

* **Framework:** React + Vite
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Deployment:** Vercel
